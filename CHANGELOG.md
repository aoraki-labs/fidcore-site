All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-21

### Added

- Initial release of FidCore website
- Landing page with the following sections:
  - Hero section with main headline and call-to-action
  - Navigation bar with responsive design
  - Solutions section showcasing company services
  - Team section introducing team members
  - Tech stack section displaying technologies used
  - Social proof section with customer testimonials
  - Footer with company information and links
- Responsive design supporting desktop, tablet, and mobile devices
- Smooth animations using Framer Motion
- Modern styling with Tailwind CSS
- TypeScript support for type safety
- Next.js 14 App Router architecture

### Technical Details

- **Framework**: Next.js 14.1.0
- **UI Library**: React 18.2.0
- **Language**: TypeScript 5.3.3
- **Styling**: Tailwind CSS 3.4.1
- **Animation**: Framer Motion 11.0.0
- **Icons**: Lucide React 0.321.0

## How to Update This File

When making changes to the project, please update this changelog:

1. Add a new version section at the top (below the header)
2. Use the following categories as needed:
   - **Added** - New features
   - **Changed** - Changes to existing functionality
   - **Deprecated** - Features that will be removed in future versions
   - **Removed** - Features that have been removed
   - **Fixed** - Bug fixes
   - **Security** - Security-related changes

3. Update the Version History Summary table
4. Follow semantic versioning:
   - **MAJOR** (x.0.0): Breaking changes
   - **MINOR** (0.x.0): New features, backward compatible
   - **PATCH** (0.0.x): Bug fixes, backward compatible

### Example Entry

```markdown
## [0.2.0] - 2025-02-15

### Added
- New contact form on the homepage
- Blog section with article listing

### Changed
- Updated navigation menu layout
- Improved mobile responsiveness

### Fixed
- Fixed broken links in footer
```
